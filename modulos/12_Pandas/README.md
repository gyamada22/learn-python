## 🎯 Objetivo
Dominar o uso do Pandas para Análise de Dados, compreendendo a estrutura de DataFrames e Series, realizando tratamento, transformação, análise, agregações e integração com arquivos CSV e Excel, preparando dados para análises avançadas e projetos reais.

---

## 📌 Conteúdos Abordados

### 🔹 Introdução ao Pandas
- O que é Pandas e para que serve
- Estrutura de Series e DataFrames
- Diferença entre Pandas e listas/dicionários Python

---

### 🔹 Importação e Exportação de Dados
- Leitura de arquivos CSV com read_csv
- Leitura de CSV diretamente da internet
- Exportação de DataFrames para CSV com to_csv
- Integração Python + Excel (Pandas vs openpyxl)

---

### 🔹 Exploração Inicial dos Dados
- info() → estrutura e tipos de dados
- head() / tail() → visualização inicial
- Identificação de valores nulos e tipos incorretos

---

### 🔹 Tratamento e Limpeza de Dados
- Remoção de colunas com drop
- Renomeação de colunas com rename
- Conversão de tipos de dados
- Padronização e organização para análise

---

### 🔹 Manipulação e Criação de Colunas
- Criação de colunas derivadas (Ano, Mês, Dia)
- Alteração de valores com condição usando loc
- Atualização de registros específicos

---

### 🔹 Filtragem de Dados
- Filtros condicionais
- Uso de loc[] e iloc[]
- Seleção de linhas e colunas específicas

---

### 🔹 Análises Estatísticas e Agregações
- value_counts() → contagem de categorias
- groupby() com:
  - sum()
  - mean()
- Ordenação de dados com sort_values
- Identificação de maiores valores com max e idxmax

---

### 🔹 Junção de Bases de Dados
- Junção de tabelas com merge
- Relacionamento entre múltiplas bases de dados
- Consolidação de dados de vendas, produtos e lojas

---

### 🔹 Visualização e Análise Exploratória
- Análises descritivas
- Apoio visual para tomada de decisão
- Preparação de dados para gráficos

---

### 🔹 Automação e Produtividade
- Barra de progresso com tqdm
- Processamento de grandes volumes de dados
- Pipelines simples de tratamento de dados

---

### 🔹 Projetos e Exercícios Práticos
- Mini Projeto de Análise de Dados
- Importação, tratamento e limpeza de dados reais
- Exercícios com:
  - GroupBy
  - Ranking de tabelas
  - Consolidação final
- Introdução ao Kaggle para prática contínua
